# 11.6 AI Agents & Autonomous Workflows

#### Problem

AI agents today operate with:

* Broad API keys
* Implicit trust
* No accountability

#### PTERI Approach

* Each agent has a cryptographic identity
* Actions require signed approval
* MCP tools enforce scope and limits

AI agents can:

* Execute payments
* Perform admin actions
* Trigger workflows

But only with **explicit, auditable authority**.
